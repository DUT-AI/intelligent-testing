# Intelligent Testing

Repository nay da duoc sap xep lai theo cau truc chuan cho du an Python + Data Science.

## Cau truc thu muc

- `src/intelligent_testing/`: ma nguon Python chinh
- `notebooks/`: toan bo notebook phan tich/thu nghiem
- `data/raw/`: du lieu goc va tai nguyen du lieu
- `docs/`: tai lieu text va ghi chu nghien cuu
- `.vscode/`: cai dat VS Code theo workspace

## Chay nhanh

```bash
python -m intelligent_testing
```

hoac sau khi cai package editable:

```bash
pip install -e .
intelligent-testing
```

## Ghi chu ve du lieu

Folder `data/raw/` chua du lieu lon (CSV/JSON/images). Hay kiem tra `.gitignore` truoc khi push de tranh day file qua lon len GitHub.
